  • Research

    Spring 2023

    Data From Docs - The Brown Institute Magic Grant Awardee

    In 2023 late spring and early fall, I helped in the grant proposal and start of Data From Docs, a 2023-2024 Magic Grant Recipient from The Brown Institute for Media Innovation (partnership between Columbia Journalism School and Stanford’s School of Engineering). The project aims to create an accessible suite of tools (OCR, entity recognition, data extraction using LLMs etc.) to help journalists, local especially, discover stories in large, unstructured sets of documents. The project tests and builds upon existing tools (such as Document Cloud and other BigLocal tools), with police misconduct data and local governing bodies' minutes collected by AgendaWatch as its evaluation dataset.

  • Coursework

    Section Leader Helper

    For CS193x: Web Programming Fundamentals, my final project was a multi-purpose tool to help with Section Leaders (SLs) as I was a section leader at the time. This includes (1) a teaching tool to create cards of their practice whiteboard, links to handout/solutions and (2) a grading tool to store comments reused for different submissions that can be viewed per homework assignment.
